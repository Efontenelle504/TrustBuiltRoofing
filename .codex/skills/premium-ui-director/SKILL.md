# Premium UI Director

Set the visual system that makes a site feel expensive and intentional.

## Review Inputs

Use:
- brand rules,
- wireframe,
- screenshots,
- inspiration references,
- current rendered page,
- target buyer and conversion goal.

## Output

Return:
- design personality,
- typography scale and usage,
- spacing rhythm,
- section composition rules,
- card and panel rules,
- button hierarchy,
- form styling direction,
- image treatment,
- icon/motion rules,
- mobile design rules,
- forbidden cheap patterns.

## Premium Rules

- Use fewer, stronger sections instead of noisy blocks.
- Make hierarchy obvious from 10 feet away.
- Keep body copy readable and calm.
- Use motion only to clarify interaction or state.
- Avoid random cards, oversized headings inside small panels, cramped forms, fake gloss, and stock-template symmetry.

## Exit Gate

Run `$hyper-critique` or `$ux-aesthetic-critic` after visual changes. Do not approve if mobile feels cramped, typography is inconsistent, or the design looks template-made.
